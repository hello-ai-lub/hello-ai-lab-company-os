- 🐼 Today's Animal（パンダ→ラッコ→メンダコ→恐竜etc.）

- 🌍 Today's Location（環境設定）
- 🎬 Today's Ending（物語の結末）

**それ以外の90%以上は固定テンプレート**です。

## 🎬 構成テンプレート（15秒固定）

```

目的: 好奇心醸成 + 短い学び

【10-15秒】PEACE + HAPPINESS PHASE
目的: 感動・癒やし・リピート欲求

例: 動物がカメラ目線で笑顔 / 最後のハグやポーズ
```
