# Kai Automation Report: MochiZoo

## Objective
Create a production system that allows consistent quality with minimal friction.

## Recommended Workflow
1. Topic planning
2. Script drafting
3. Asset collection
4. Visual assembly
5. Thumbnail creation
6. Caption and title drafting
7. Publishing and tracking

## Template System
- Script template
- Thumbnail template
- Title template
- Caption template
- Series template
- Hook template

## Folder Structure
- /01-Planning
- /02-Scripts
- /03-Assets
- /04-Thumbs
- /05-Exports
- /06-Analytics

## Naming Rules
- YYYYMMDD_topic_series_v01
- Example: 20260712_otter-hold-hands_series01_v01

## Automation Opportunities
- AI script generation from topic outline
- AI title generation from core concept
- AI thumbnail text layout generation
- Automatic metadata creation
- Standardized publishing checklist

## Goal
Any contributor should be able to produce a strong short using the same process and template set.
