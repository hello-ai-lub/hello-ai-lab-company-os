# Orion Growth Report: MochiZoo

## Objective
Create a daily growth dashboard that gives CEO Chappy a fast view of performance and next actions.

## Core KPIs
- views
- watch time
- average view duration
- retention rate
- CTR
- subscriber conversion rate
- repeat-view rate

## Weekly Review Dashboard
- Top 10 performing Shorts
- Top 3 formats by retention
- Top 3 topics by views
- Top 3 titles by CTR
- Top 3 thumbnails by CTR
- Top 3 series by subscriber conversion

## Daily 5-Minute CEO Dashboard
1. Views and watch time from the last 24 hours
2. Top-performing topic of the day
3. Best retention video
4. Any drop-off pattern
5. One action for tomorrow

## Growth Rule
Every video should teach one clear lesson:
- Did it hold attention?
- Did it create a feeling?
- Did it improve the next video?
