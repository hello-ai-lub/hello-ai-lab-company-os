## 📚 セクション2：各項目の詳細説明 & 最適な書き方

### **1. CHARACTER_DESCRIPTION（キャラクター描写）**

**目的**: AIが生成するキャラの見た目を完全に制御

**最適な記述方法**:
```
❌ 悪い例
cute animal

✅ 良い例
fluffy white rabbit with round proportions, big sparkly eyes, pink nose, 
soft fur texture, kawaii proportions (large head, small body)
```

**使える表現**:
- `fluffy` = ふわふわ
- `round proportions` = 丸いプロポーション
- `sparkly eyes` = キラキラした目
- `kawaii anime style proportions` = 萌え系プロポーション
- `soft-featured, charming look` = 優しい顔立ち

**長さ目安**: 20-40単語

---

### **2. KEY_EXPRESSION（表情指定）**

**目的**: キャラが何を感じているのかを明確に

**パターン集**:
```
Happy/Cheerful:
- "bright smile with sparkling eyes"
- "joyful, beaming with happiness"
- "cute giggle expression"

Curious/Inquisitive:
- "tilted head, eyes wide with wonder"
- "playfully curious, slightly tilted smile"
- "thoughtful gaze with sparkle"

Surprised/Delighted:
- "wide eyes with mouth open in wonder"
- "thrilled expression, eyebrows raised"
- "surprised and delighted simultaneously"

Calm/Peaceful:
- "gentle smile, peaceful eyes"
- "serene expression, soft gaze"
- "content and dreamy look"
```

**長さ目安**: 10-15単語

---

### **3. MOVEMENT_DESCRIPTION（動き）**

**目的**: 動画として生きた動きを表現

**テンプレート式**:
```
[身体部位] + [速度] + [品質形容詞]

例：
"gentle head tilt with smooth, naturalistic motion"
"playful hop with bouncy, light-on-feet energy"
"slow elegant turn with flowing, dreamlike quality"
"expressive paw/hand gestures with precise, purposeful movement"
"subtle ear wiggle with cute, twitchy charm"
"smooth glide movement, as if floating on air"
```

**Kling AIに効果的な動詞**:
- `gently` = 優しく
- `smoothly` = スムーズに
- `playfully` = 遊び心を持って
- `elegantly` = 優雅に
- `bouncy` = はずむような
- `fluid` = 流動的な

**長さ目安**: 15-25単語

---

### **4. LOCATION_DESCRIPTION（背景・ロケーション）**

**目的**: 世界観とシーンの雰囲気を統一

**パターン集**:
```
Nature:
"sunny meadow with vibrant wildflowers, white clouds, soft grass"
"enchanted forest with magical glowing elements, ethereal mist"
"serene beach with golden sand, turquoise water, palm trees"

Urban/Modern:
"cozy apartment with warm interior design, soft furniture"
"whimsical toy store with colorful displays and shelves"
"modern studio with minimalist aesthetic, clean lines"

Fantasy/Imaginary:
"magical candy kingdom with pastel colors, cloud platforms"
"dreamy space environment with floating planets and stars"
"enchanted garden with bioluminescent plants, magical atmosphere"
```

**重要**: 可愛い世界観であることを強調
```
"cute, inviting atmosphere" OR "whimsical, child-friendly environment"
```

**長さ目安**: 20-35単語

---

### **5. TIME_OF_DAY（時間帯・光源）**

**目的**: 光の質と色温度を制御

**設定別のプロンプト**:
```
Golden Hour (朝焼け/夕焼け):
"golden hour sunset, warm amber light, long shadows, magical glow"

Bright Daytime:
"sunny day, bright natural daylight, clear visibility, vibrant colors"

Soft Overcast:
"overcast day, diffuse soft light, no harsh shadows, gentle atmosphere"

Magical Night:
"twilight hour, soft moonlight, magical glow, bioluminescent accents"

Studio Light (人工光):
"studio lighting setup, perfectly balanced 3-point lighting, professional"
```

**長さ目安**: 8-20単語

---

### **6. LIGHTING_STYLE（ライティング・光源技術）**

**目的**: Kling AIに最高品質の照明を指示

**基本パターン**:
```
必須要素: [光源] + [質感] + [品質形容詞]

例：
"Professional 3-point studio lighting with key light, fill light, 
and rim light creating depth and dimension, perfectly balanced for maximum visual appeal"

"Warm cinematic lighting with soft shadows, volumetric light rays, 
creating intimate and magical atmosphere"

"Backlighting with golden rim light, front-lit with soft key light, 
creating dimensional, theater-quality illumination"
```

**Kling AI最適化表現**:
- `volumetric lighting` = ボリュメトリック照明
- `rim lighting` = リムライト
- `key light + fill light` = キー・フィル照明
- `soft shadows` = やさしい影
- `cinematic contrast` = 映画的コントラスト
- `ray tracing quality` = レイトレーシング品質

**長さ目安**: 30-50単語

---

### **7. CAMERA_MOVEMENT（カメラワーク）**

**目的**: 動画のダイナミズムを制御

**YouTube Shorts向け最適パターン**:
```
ショート動画（12-15秒）は「1つのカメラムーブ」推奨

例：
"Static wide shot showing full scene, slight subtle camera tilt upward"
"Slow dolly-in shot moving closer to subject over 10 seconds"
"Gentle pan camera movement following character's motion"
"Smooth tracking shot moving around the subject in circular motion"
"Slight zoom with floating feel, as if gravity is reduced"
```

**ショート動画で避けるべき**:
- ❌ 急激な動き
- ❌ 複数のカメラムーブ混在
- ❌ 不安定な揺れ

**長さ目安**: 15-30単語

---

### **8. LENS_TYPE（レンズ・焦点距離）**

**目的**: 視覚的な圧縮感や透視感を制御

**設定別表現**:
```
Wide/Cinematic:
"cinematic wide angle lens (35mm equivalent), capturing full scene"

Intimate/Close:
"medium telephoto lens (85mm equivalent), creating depth compression, 
subject in focus, dreamy background blur"

Macro/Detail:
"macro lens perspective, intimate close-up view of fine details"

Surreal/Artistic:
"anamorphic lens style, cinematic character-to-width ratio, 
artistic perspective distortion"
```

**最高品質フレーズ**:
```
"Premium cinema lens, razor-sharp focus, buttery background bokeh, 
professional optics quality"
```

**長さ目安**: 10-20単語

---

### **9. TEXTURE_DESCRIPTION（質感）**

**目的**: キャラクターと世界の物質感をリアルに

**質感パターン**:
```
キャラクター用:
- `soft fur with fine detail` = 毛並みディテール
- `smooth skin with subtle highlights` = つるつる肌
- `plush, velvety surface` = ぬいぐるみのような
- `crystalline, translucent quality` = 透明感

環境用:
- `lush grass with fine blade detail` = 詳細な草
- `weathered wood texture, organic imperfections` = 自然な木肌
- `smooth ceramic, clean surface` = つるつるした表面
- `fabric with weave visible` = 布地の織目
```

**Kling AI推奨表現**:
- `photorealistic texture` = フォトリアル質感
- `microscopic detail level` = 微細ディテール
- `professional material library` = 高品質マテリアル

**長さ目安**: 15-25単語

### **10. MOOD_DESCRIPTION（空気感・雰囲気）**

**目的**: 動画全体の感情的トーン

**子ども向け・可愛い系**:
```
"Joyful, wholesome atmosphere, safe and friendly environment,
```

**YouTube Shorts向け**:

**長さ目安**: 20-35単語

### **11. COLOR_PALETTE（色味）**

**目的**: 統一された色彩イメージ

**テンプレート式**:
```
[メイン色] + [アクセント色] + [全体トーン]

例：
"Warm pastel palette with soft pinks, mint greens, and creamy whites; 
sunny golden accents creating cheerful, optimistic mood"

"Cool magical tones with purples, blues, and turquoise; 
glowing pink highlights adding whimsical energy"

"Earthy natural colors with warm browns, soft yellows, and green accents; 
sunset orange creating magical warmth"

"Vibrant candy-colored palette with bright but not harsh colors; 
saturated but sophisticated, appealing to children"
```

**Kling AI最適化のコツ**:
- 「warm」「cool」「vibrant」「soft」などの形容詞を追加
- 3-4色の調和を明示
- 全体トーン（cheerful, magical, serene等）を最後に記載

**長さ目安**: 25-40単語

### **12. NEGATIVE_PROMPT（ネガティブプロンプト：不要な表現）**

**目的**: AIが生成してはいけない要素を明確に指示

**必須ネガティブプロンプト**:
```
ALWAYS INCLUDE:

low quality, blurry, pixelated, oversaturated, undersaturated, 
harsh lighting, flat lighting, amateur, unfinished, draft, sketch, 
distorted face, wrong hands, deformed anatomy, uncanny valley, 
creepy expression, sad or angry character, scary elements, 
violence, dark atmosphere, murky colors, watermark, text overlay, 
unnatural proportions, unrealistic anatomy, stiff movement, 
floating artifacts, glitches, digital noise, compression artifacts
```

**MocchiZoo向け追加NG**:
```
NOT FOR CUTE CONTENT:

realistic human skin, photorealistic human, 
overly detailed animal anatomy, grotesque features, 
exaggerated Disney realism, 3D render artifacts, 
CGI plastic look, uncanny proportions, 
adult themes, sophisticated/mature style, 
grayscale or desaturated colors, horror elements
```

**テンプレート内での使用法**:
```

[既存の必須NG] + [コンテンツ固有のNG]
```

## ⚙️ セクション5：Hello AI Lab 推奨設定

### **Kling AI 標準パラメータ**

| パラメータ | 推奨値 | 理由 |
|-----------|--------|------|
| **Model** | Kling 1.5 (最新版) | 最高品質・可愛さ表現が強い |
| **Video Size** | 9:16 (1080×1920) | YouTube Shorts最適（100%推奨） |
| **Duration** | 15 seconds | MocchiZoo標準・最高品質達成可能 |
| **Quality** | Premium/High | 8K出力対応・描写が細かい |
| **CFG Scale** | 7.5-8.5 | プロンプト従順性バランス |
| **Seed** | Random | 多様性・新鮮さ |

### **Kling生成の試行フロー（推奨）**

```
第1試行（ベース確認）
├─ テンプレートプロンプトをそのまま使用
├─ パラメータは推奨値
└─ 所要時間：5-10分

結果判定:
├─ ✅ 80点以上 → 微調整試行へ
├─ ⚠️ 60-80点 → プロンプト修正試行へ
└─ ❌ 60点以下 → ベースプロンプト検証

第2試行（プロンプト最適化）
├─ 不足要素を追加（例：色が不足なら色指定追加）
├─ 曖昧な表現を具体化
└─ 品質形容詞を積み重ね

第3試行（微調整）
├─ キャラクターの細部修正
├─ 背景の改善
└─ ライティングの調整

MAX試行回数：3回
理由：品質はプロンプト改善3回で頭打ち。
      以降は設定値変更（CFG, Seed, Model変更）を検討
```

### **トラブルシューティング**

| 問題 | 原因 | 解決策 |
|------|------|--------|
| キャラが可愛くない | 表情指定不足 | `big sparkly eyes`, `charming smile` を追加 |
| 背景が不自然 | ロケーション記述が曖昧 | 具体的な場所+雰囲気を明記 |
| 色が暗い | 時間帯指定なし | `golden hour` または `bright daylight` を明示 |
| 動きが固い | 動き説明が抽象的 | `smooth`, `flowing`, `gentle` などを追加 |
| CGっぽい | 品質指定が弱い | `Pixar-quality`, `cinematic` を強調 |
| 15秒完成しない | Kling AIの処理限界 | 動きを削減、静止シーンを増加 |
