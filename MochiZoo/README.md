# MochiZoo

## 事業概要
MochiZooは、AI社員チームによってYouTubeコンテンツを企画・制作・投稿・分析する実験的な事業です。

## 事業の特徴
- YouTubeをAIチームで運営する実験である
- 企画から投稿、分析、改善までを一連の流れとして扱う
- AI社員の役割分担を実際に試す場である

## 現在の進捗
- Hello AI Labの基本組織を整理した
- TAKEとチャッピーの役割を明確にした
- MochiZooの運営フローを整理した
- 初期の役職候補を整理した

## 次に決めること
- まず最初に採用するAI社員の役職
- MochiZooのコンセプトの方向性
- 企画から投稿までの初期ルール
