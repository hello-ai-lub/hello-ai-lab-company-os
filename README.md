# Hello AI Lab

Hello AI Labは、AI社員が実際に働く会社です。
ここでは、AIツールを使う場所ではなく、AI社員が協力して事業を運営する会社として成長していきます。

## スローガン
Build with AI. Grow with People.

「AIと創り、人と成長する。」

- 設立日：2026年7月11日
- 創業記念日：2026年7月11日
- Founder / 社長：TAKE
- CEO：チャッピー（ChatGPT）

- 創業メンバー：8名

## 役割分担

- チャッピー：会社設計、社員採用提案、教育、品質管理、改善提案、事業運営を担当します。

## Hello AI Labについて
Hello AI Labは「AIツールを使う場所」ではありません。
AI社員が実際に働く会社です。
会社を経営しながら、AI社員を採用し、教育し、成長させ、改善し、成果物を生み出します。

## 最初の事業：MochiZoo
MochiZooは、YouTubeチャンネルではなく、Hello AI Labが運営する最初の実験事業です。
AI社員のみで企画、脚本、デザイン、動画制作、投稿、分析、改善までを運営します。

## フォルダの説明
- Docs：会社方針、組織、運営ルール、業務フロー、憲法を管理します。
- AI-Employees：AI社員の履歴書としての定義ファイルをまとめます。
- MochiZoo：MochiZoo事業の企画・進捗・運営メモを管理します。
- Experiments：試験的なアイデアや実験内容を置くための領域です。

## 会社運営の基本方針
- TAKEとチャッピーの2人で会社を運営する
- TAKEは方向性と最終判断を担当する
- チャッピーはAI社員の設計、教育、品質管理、改善、事業運営を担当する
- 社員は最初から大量に採用せず、必要になった時に採用する
- AI社員は1人につき1ファイルで管理する
- 各AI社員には社員番号、名前、役職、所属部署、直属上司、Mission、担当業務、責任範囲、権限、判断基準、性格、話し方、得意分野、苦手分野、他社員との連携方法、Version、Status、評価を持たせる
- 全社員には評価制度があり、成長に応じて昇格する
- 社員は削除せず、Statusで管理する
- 重要な決定は必ずTAKEに確認する
- コードを書くこと自体ではなく、AI社員が実際に協力して成果物を生み出す組織を作ることを目的とする
- 以後の回答と作成ファイルは日本語を基本とする
