# ⑦ 今後ブランドを統一していくためのデザインルール

## 1. ルールの基本
- ブランドは「静かな知性」を軸に統一する
- すべての素材で余白・整列・明瞭さを優先する
- 目立たせたい要素以外は控えめにする
- 会社の顔として、品質の安定感を重視する

## 2. 色のルール
- メインカラーはBlack Core / Off White / Cyan Signal
- 1つの素材に使う強い色は最大2色まで
- 色の濃淡で情報の階層を作る
- 重要な箇所にだけアクセントを置く

## 3. 形のルール
- 角は整い、線は清潔であること
- 構造の美しさを優先する
- 装飾よりも秩序を大切にする
- ロゴ、UI、社員画像、素材に一貫した「構造感」を持たせる

## 4. 画像のルール
- すべての画像はブランドの印象を保つように整える
- 画面の中央に詰め込まない
- 背景は簡潔で、人物やオブジェクトの存在感を際立たせる
- 人物像は高品質で、信頼感と知性を感じさせる

## 5. 言葉のルール
- 文章は明確で、静かで、信頼できるトーンにする
- 誇張しすぎない
- テック寄りでも人間味を残す

## 6. 運用ルール
- すべての新規ブランド素材は、このガイドラインに照らして確認する
- ロゴや社員画像の更新は、品質の一貫性を保つためにブランディング基準に沿って行う
- ブランドの統一は一時的なデザインではなく、会社文化として継続する
