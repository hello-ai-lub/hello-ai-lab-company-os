# Hello AI Lab Company Dashboard v2

## 概要
このダッシュボードは、Founder TAKE、CEO チャッピー、AI社員が会社の状況を確認するための社内管理画面です。
Version 2では、より高級感のある黒基調UI、社員プロフィールの詳細表示、部署のカラー整理、進捗バー、会議予定、CEOからの指示などを追加しました。

## いちばん簡単な起動方法
1. [Dashboard/START-DASHBOARD.command](START-DASHBOARD.command) を見つける
2. そのファイルをダブルクリックする
3. もし許可の確認が出たら「Allow」を押す
4. 自動でブラウザが開いて http://localhost:8000/ を表示する

## もし最初に許可が出たら
- 「Allow」を押してください
- それだけで起動します

## 手動で起動する方法
ターミナルが使える場合は、次のコマンドでも起動できます。

```bash
cd Dashboard
python3 -m http.server 8000
```

その後、ブラウザで http://localhost:8000/ を開きます。
